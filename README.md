## 第一次請安裝

```bash
1. NODJS下載後並安裝 https://nodejs.org/
2. 下載程式並解壓縮: https://github.com/RyanRoll/inn-crawler/archive/refs/heads/master.zip
3. 使用命令字元進入該資料夾 inn-crawler
4. 執行 $ npm install
```

## 執行爬蟲

```bash
$ npm start

// or sel_area=27, sel_area_txt=長野, intervalTime=1H

$ npm start npm start 27 長野 3600
```
